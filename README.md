# Comfyui-Jojoagent-ai-V2

Jojoagent V2 是一套面向 ComfyUI 的图像生产辅助插件，提供提示词优化、识图反推、电商详情页提示词、多图批量生成和用量查看等节点。

插件适合用于商品图、详情页、画册、人像视觉、模特换装、风格延展等工作流。

## 节点介绍

### Jojo V2 提示词优化器

用于把简单需求、参考图和基础指令整理成更完整的图像生成提示词。

主要能力：

- 支持最多 5 张参考图
- 支持模板选择，如产品视觉、高级画册、人像摄影视觉等
- 支持模式选择，用于控制统一视觉或多方向创意
- 可按屏次数量生成多条连续提示词
- 输出单条优化提示词和提示词列表，方便继续连接图像生成节点

### Jojo V2 识图提示词反推

用于分析参考图，并反推出适合再次生成或二次创作的提示词。

主要能力：

- 支持多图输入
- 可输出画面分析、正向提示词和负向提示词
- 适合用于图片复刻、风格迁移、商品图重绘、参考图拆解
- 可选择不同识图模型，以适配不同的分析需求

### Jojo V2 电商详情页提示词

用于根据商品图、风格参考和卖点信息生成电商详情页分屏提示词。

主要能力：

- 支持正面图、背面图、侧面图、风格参考图输入
- 可填写商品名称、核心卖点、材质细节和使用场景
- 可设置生成屏数
- 输出多屏提示词列表，方便直接连接批量图像生成节点
- 适合详情页首屏、卖点屏、材质屏、场景屏、对比屏等场景

### Jojo V2 AI 批量图像生成

用于根据单条提示词或提示词列表批量生成图片。

主要能力：

- 支持最多 5 张参考图
- 支持单提示词生图和多提示词批量生图
- 支持图片规格、比例、质量和随机种子设置
- 输出图片和批量生成信息
- 可与提示词优化器、详情页提示词节点联动

### Jojo V2 扣费详情

用于查看当前账号最近的使用记录。

主要能力：

- 显示当前余额
- 显示最近记录的模型名称和扣除金额
- 可设置显示条数
- 可筛选只看成功记录

## 安装方法

把整个文件夹放入 ComfyUI 的自定义节点目录：

```text
ComfyUI/custom_nodes/
```

例如：

```text
ComfyUI/custom_nodes/Comfyui-Jojoagent-ai-V2/
```

然后重启 ComfyUI。

## 基础配置

每个节点中填写：

```text
api_base = 服务地址
api_token = 你的账号口令
```

`api_token` 请使用服务方提供的账号口令。

## 常见问题

### 节点没有出现

请确认插件文件夹放在：

```text
ComfyUI/custom_nodes/
```

然后完全重启 ComfyUI。

### 模型列表没有更新

如果旧工作流中仍显示旧选项，删除旧节点后重新添加一次新节点。

### 提示连接失败

请检查：

- `api_base` 是否填写正确
- `api_token` 是否填写正确
- 本机网络是否可以访问服务地址
- 服务端是否正常运行
