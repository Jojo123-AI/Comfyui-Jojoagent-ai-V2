# Jojoagent RH V2 客户扣费公示

本插件通过 Jojoagent 额度服务器调用 RunningHub 上游模型。客户实际扣费按以下规则计算：

```text
客户扣除积分 = (模型扣费 + 用时扣费) x 1.3
```

最终扣费以服务器记录为准。

## 用时扣费

| 运行机型 | 上游单价 | 折算方式 |
| --- | ---: | --- |
| Lite | 0.4 / 小时 | `0.4 x 运行秒数 / 3600` |
| Standard | 4 / 小时 | `4 x 运行秒数 / 3600` |
| Plus | 6 / 小时 | `6 x 运行秒数 / 3600` |

## 示例

如果使用 `gpt-image-2-低价` 生成 1K 图片，运行机型为 Lite，任务耗时 90 秒：

```text
模型扣费 = 0.16
用时扣费 = 0.4 x 90 / 3600 = 0.01
客户扣费 = (0.16 + 0.01) x 1.3 = 0.221
```
